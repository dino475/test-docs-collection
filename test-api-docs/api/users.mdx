# Users API

Manage user accounts and profiles.

## Get User

```http
GET /api/users/{id}
```

### Response

```json
{
  "id": "123",
  "name": "John Doe",
  "email": "john@example.com"
}
```

## Create User

```http
POST /api/users
```

### Request Body

```json
{
  "name": "Jane Doe",
  "email": "jane@example.com"
}
```
