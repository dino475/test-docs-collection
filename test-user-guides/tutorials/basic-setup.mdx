# Basic Setup Tutorial

Learn how to configure the essential settings for your account.

## Configuration Overview

This tutorial covers:
- Account preferences
- Notification settings  
- Basic integrations

## Account Preferences

### Personal Information
Update your profile with:
```
Name: John Doe
Email: john@company.com
Time Zone: UTC-5 (Eastern)
```

### Display Settings
Choose your preferred:
- Theme: Light or Dark mode
- Language: English, Spanish, French
- Date format: MM/DD/YYYY or DD/MM/YYYY

## Notification Settings

Configure how you receive updates:
- **Email notifications**: Daily digest or real-time
- **Browser notifications**: Enable for important alerts
- **Mobile push**: Download our app for mobile alerts
